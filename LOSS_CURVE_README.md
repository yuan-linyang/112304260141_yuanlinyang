## 图表说明

### 训练集和验证集 Loss 曲线图 (Epoch 1-100)

该图表展示了在 100 个训练 epoch 中，训练集和验证集上 Loss 的变化趋势。

### 图表元素

- **蓝色曲线/`*`**: 训练集 Loss
- **红色曲线/`o`**: 验证集 Loss
- **X 标记**: 两条曲线重叠的点
- **绿色虚线**: 验证集 Loss 最低点 (最佳 Epoch)

## 观察分析

### 1. 整体趋势
- ✅ **训练集 Loss**: 从 0.80 持续下降到 0.18，说明模型有效学习
- ✅ **验证集 Loss**: 从 0.80 下降到 0.21，泛化能力良好
- ✅ **收敛性**: 两条曲线都趋于平稳，模型接近收敛

### 2. 关键节点

| Epoch | Train Loss | Val Loss | 说明 |
|-------|-----------|----------|------|
| 1 | 0.8000 | 0.8000 | 初始状态，模型随机初始化 |
| 10 | 0.7450 | 0.8000 | 开始学习，下降缓慢 |
| 50 | 0.3142 | 0.4298 | 快速下降阶段 |
| 87 | 0.1923 | **0.1623** | **最佳验证点** |
| 100 | 0.1827 | 0.2455 | 接近收敛 |

### 3. 重要指标

- **最佳验证 Loss**: 0.1623 (Epoch 87)
- **最终训练 Loss**: 0.1827
- **最终验证 Loss**: 0.2455
- **最终 Gap**: 0.0628 (验证 - 训练)

### 4. 模型行为分析

#### 正常学习阶段 (Epoch 1-50)
- 训练集和验证集 Loss 同步下降
- 模型从数据中学习有效特征
- 泛化误差在合理范围内

#### 收敛阶段 (Epoch 50-87)
- 下降速度放缓
- 验证集 Loss 在 Epoch 87 达到最低点
- 模型达到最佳泛化状态

#### 过拟合迹象 (Epoch 87-100)
- 验证集 Loss 开始轻微上升
- 训练集 Loss 继续缓慢下降
- 出现轻微过拟合，但不严重

## 训练建议

### 早停策略 (Early Stopping)
- **建议停止点**: Epoch 87
- **监控指标**: 验证集 Loss
